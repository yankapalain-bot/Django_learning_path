📚 Key Database Concepts in This App
PK Primary Key

Every model has an auto-generated id field — the unique identifier for each row. Django sets this automatically.

FK Foreign Key

ForeignKey(Model) stores the PK of another table's row. Example: Neighbourhood.city_id stores a City's PK.

1:1 OneToOne

OneToOneField is a FK + UNIQUE constraint. Each Restaurant links to exactly one PointOfInterest and vice-versa.
